Shortcodes with visitor's info. Also enables usage of conditional content.

== Description ==

The plugin enables user to add various data from visiting user in the content as a shortcode.

== Installation ==

1. Upload `personalized-shortcode-pro.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
